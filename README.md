# Solo Leveling — Workout Tracker

Ton app de suivi d'entraînement style Solo Leveling.

## Déploiement sur Vercel (5 min)

### 1. Crée un repo GitHub
- Va sur github.com → New repository
- Nomme-le `solo-leveling-workout`
- Laisse tout par défaut → Create repository

### 2. Upload les fichiers
- Clique "uploading an existing file"
- Glisse tous les fichiers du zip (en conservant la structure)
- Commit changes

### 3. Déploie sur Vercel
- Va sur vercel.com → Continue with GitHub
- New Project → Importe `solo-leveling-workout`
- Clique Deploy → c'est tout ✅

### 4. Ajoute sur l'écran d'accueil iPhone
- Ouvre ton URL Vercel dans Safari
- Icône Partager → "Sur l'écran d'accueil"
- C'est comme une vraie app 📱
